Los permisos de cámara y micrófono se gestionaron mediante la API getUserMedia del navegador, solicitándolos a través de un botón y mostrando su estado al usuario. Al concederlos, se habilita la vista previa de la cámara, la captura de una imagen y la grabación de audio durante cinco segundos. Para la integración con una API externa se utilizó un análisis de sentimiento de texto, ya que permite procesar fácilmente la información ingresada por el usuario. Se envía una descripción escrita sobre la captura realizada y se recibe como respuesta la etiqueta del sentimiento y su nivel de confianza, los cuales se muestran en pantalla.
